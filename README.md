# rpi-centos-aarch64

## chainsx又一大作

## 前言

对于RaspberryPi3而言，centos官方给的支持仅有armhf，而armhf的软件包屈指可数，连桌面都不行（反正我没安装成功过）。

现在对于aarch64的主流启动方式为UEFI，例如openSUSE，但是我个人能力不够，没办法╮(╯_╰)╭，只好用传统启动方法。

## 使用说明

默认用户为`root`，密码`centos`。

第一次开机不会拓展根目录，请自行拓展。

## 关于

内核（kernel）：linux-4.10.y

根目录（rootfs）：官方centos7 aarch64
